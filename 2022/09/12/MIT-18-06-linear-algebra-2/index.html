1. Elimination 的基本思想

上面是一个简单的 2 by 2 linear system，我们要得到这个 system 的解就要用到 elimination。

  1. 首先我们需要确定这个 system 的 pivot，一般是第一行的第一个元素，在这里是1
  2. 然后求得需要放大的倍数E21 = (Entry to eliminate in row 2) / (pivot in row 1) = 3 / 1 = 3
  3. 我们把第一个方程各项乘以E21与第二个方程组合，得到下面的 system
  4. 把第二个方程与第一个方程相减后再与第一个方程组合，得到下面的 system

至此，我们把一个 linear system (Ax=b) 变成了一个 upper triangular (Ux=c)
由8y=8可以得出y=1，再把y=1代入第一个方程，得出x=3。
所以x=3,y=1就是这个 linear system 的解
