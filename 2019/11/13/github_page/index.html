利用 Github Pages 搭建博客

Posted on 2019-11-13

介绍如何利用 Github Pages 搭建博客

一、Jekyll

Github Pages 使用的 Jekyll

  1. Jekyll 安装
    1. Jekyll 基于 Ruby，首先应安装 Ruby 环境
      sudo apt install ruby
      sudo apt install ruby-dev
    2. 使用 gem 安装 jekyll 和 bundle
      gem install jekyll bundle
  2. 到网上下载 Jekyll 主题，按照 readme 去安装使用
  3. 若运行期间提示错误，可尝试用 gem uninstall package_name 卸载相应的包后，使用 gem install install package_name，安装最新的包

二、Github Pages

创建仓库，仓库名必须是 “user_name.github.io”，完成。

三、Git 同步

配置 .ssh/.config 添加 ssh key

测试配置是否成功

ssh -T git@github.com
