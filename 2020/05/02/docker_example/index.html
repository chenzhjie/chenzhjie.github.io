Docker CE 的使用简介

一、Docker 安装

  1. Uninstall old versions
    sudo apt-get remove docker docker-engine docker.io containerd runc

  2. Install using the repository

    1. sudo apt update

    2. sudo apt install apt-transport-https ca-certificates curl gnupg-agent software-properties-common

    3. Add Docker’s official GPG key:
      curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

    4. verify the fingerprint 9DC8 5822 9FC7 DD38 854A E2D8 8D81 803C 0EBF CD88
      sudo apt-key fingerprint 0EBFCD88

  3. Install Docker Engine - Community
    sudo apt update
    sudo apt install docker-ce docker-ce-cli containerd.io

  4. Install a specific version of Docker Engine
    apt-cache madison docker-ce

     sudo mkdir -p /etc/docker
 sudo tee /etc/docker/daemon.json <<-'EOF'
 {
     "registry-mirrors": ["https://ivri9wbi.mirror.aliyuncs.com"]
 }
 EOF

    3. Reload config file
    `sudo systemctl daemon-reload`
    `sudo systemctl restart docker`

  5. Docker example

    1. nextcloud
      docker run -d --name nextcloud -p 80:80 -v /root/nextcloud:/data rootlogin/nextcloud
    2. wukong
      docker run -it -p 5000:5000 --device /dev/snd -e LANG=C.UTF-8 wzpan/wukong-robot:latest
