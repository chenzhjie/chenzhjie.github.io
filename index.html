Kindle 越狱并改造为日历

Posted on 2021-02-09

1. 为什么要越狱

在开始之前先说一下为什么要越狱

  1. 越狱之后可以得到什么 :
    越狱之后就是一个 Linux 系统，Linux 能做的事它都能做
  2. 越狱之后的缺点 :
    与此同时，越狱之后作为一个阅读器的体验可能会下降，包括耗电量可能会增加。

所以，如果你要越狱，一般的情况是你有多个 Kindle，或者你的 Kindle 长时间闲置不用，
需要把它改造成其他东西，也可以是单纯想折腾一下

2. 如何越狱

特定版本的 Kindle 可以利用系统漏洞越狱，看自己的 Kindle 是否符合条件，请到下面网址查询：Kindle 越狱教程：利用特制固件越狱
如果你的 Kindle 版本不符合上面的要求，就需要拆机利用串口进行破解。

3. 拆机越狱的方法

我只会大概说一下我在越狱过程中觉得需要注意到点，具体的操作请参考下面网址：Kindle 7 拆機硬件調試串口越獄教程，這或許是中文互聯網唯三的拆機越獄教程！

1. 拆机。

  1. Kindle 的前面板是用胶水粘着的，需要先把前面板撕下来，才能看到里面的螺丝。

    注意：撕的时候注意，不要把屏幕上贴的保护膜破坏，还有最下面的 LED 灯不要弄坏了，可以用
    热吹风来拆
  2. 把螺丝拧下来后就可以看到主板了
  3. 主板上印丝写着 Serial Debug 的就是串口，TX 和 RX 的位置可能每个版本的 PCB 都不一样。我的是 TX, RX, GND，具体情况要自己测试

    注意：这个串口是1.8V，要判断自己用的 TTL2USB 模块是否支持1.8V。

2. 计算 root 密码。

登录该网站，输入序列号，计算 root 密码。

3. 使用 Putty 登录控制台

  1. 在控制台上显示按任何键进入 uboot 时，按任何键进入 uboot，按照上面的教程选择项目，利用串口登录控制台，用户是 root，密码是上面计算的结果
  2. 删除 root 密码，并重启
  3. 重启后进入系统，下载 jailbreak 文件后放到 Kindle 盘符的根目录，拔掉数据线，在控制台中输入:
    cd /mnt/us && ./jb.sh
    执行完成后，屏幕的下方会显示 jailbreak
  4. 下载 hotfix 文件，放到 kIndle 盘符的根目录，在 Kindle 菜单中选择系统更新，等待更新完成，Kindle 会自动重启。至此，你的 Kindle 已经越狱成功

4. 安装插件。

  1. 安装 KUAL 和 MPRI，这两个插件安装后，就可以安装其他插件了
  2. 可以安装插件推荐:Kindle 越狱插件资源下载及详细安装步骤

5. Kindle 日历。
